# 🎵 HERTZ

> A music streaming web app with AI-powered recommendations, playlist creation, and song reviews.

HERTZ is yet being developed however most of the front-end part is completed

---

## Features

- **AI Recommendations** — Personalized song and artist suggestions that adapt to your listening history
- **Playlist Creation & Sharing** — Build playlists and share them with a public link
- **Song Reviews & Scores** — Rate and review tracks; see aggregated community scores on every song

---

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/)

---

## Prerequisites

- Next.js v16+
- Node.js v19+
- npm
- A `.env.local` file with the required environment variables (see [Configuration](#configuration))

---

## Installation

```bash
# Clone the repo
git clone https://github.com/your-username/hertz.git

# Move into the project directory
cd hertz

# Install dependencies
npm install
```

---

## How to run

```bash
# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---
